Instructions:

Before you begin:
  • Ensure that you have imported master data from Data manager.
To import demo data:
  1. Download DemoDataStep1.zip, which contains sample files for the import of clause categories, contract types, parties, external contacts, organizations, and terms.
    ZIP file Contract data included
    DemoDataStep1.zip
    • ClauseCategory
    • ContractType
    • Party
    • ExternalContact
    • Organization
    • Term
  2. Download DemoDataStep2.zip, which contains sample files for the import of clauses, account numbers, account managers, contact mappings and members.
    ZIP file Contract data included
    DemoDataStep2.zip
    • Clause
    • AccountNumber
    • AccountManager
    • ContactMapping
    • Member
  3. In DemoDataStep1.zip and DemoDataStep2.zip, replace the required user-related values in the Microsoft Excel files to match the values specified in OpenText Directory Services (OTDS).
  4. To import contracts with templates, create the required templates manually.
  5. Download DemoDataStep3.zip, which contains sample files for the import of contracts.
    ZIP file Contract data included
    DemoDataStep3.zip
    • Contract
  6. In DemoDataStep3.zip > Contract.xls, modify the template values as required.
  7. On the Upload tab, upload the modified ZIP files in the order they are listed:
    • DemoDataStep1.zip
    • DemoDataStep2.zip
    • DemoDataStep3.zip
    Note: Do not change the name of the files before upload.
